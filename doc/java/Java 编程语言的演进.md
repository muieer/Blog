1、本地变量类型推断

3、switch 表达式
4、Record Class
5、模式匹配（Pattern Matching）
6、字符串模板（String Templates）
7、Unnamed Classes and Instance Main Methods(写小片段程序)

1、Sealed Classes

## 好处
- 因为简化了编写，所以写出的代码易读，因为易读写出的程序更容易维护
- 因为更健壮，不容易出现人为的错误，程序更安全和好维护
